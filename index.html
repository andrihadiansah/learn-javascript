Introduction

JavaScript is a versatile and powerful programming language primarily used to create interactive and dynamic content on the web. This repository will guide you through its fundamental concepts.

Topics Covered

1. Basic Concepts

Variables

In JavaScript, there are three ways to declare a variable: var, let, and const.

  • var: Declares a variable globally or locally to an entire function regardless of block scope.
  • let: Declares a block-scoped variable, meaning it is only available within the block it is declared.
  • const: Declares a block-scoped, read-only constant variable that cannot be reassigned.
Functions

A function in JavaScript is a block of code that performs a specific task and can be executed when called.

  • Function Declaration
  • Function Expression
  • Arrow Functions
Loops

Loops are used to repeat a block of code multiple times. JavaScript provides several types of loops:

  • for: Executes a block of code a specific number of times.
  • while: Executes a block of code as long as the condition is true.
  • do...while: Executes the block of code once, and then continues while the condition is true.
Conditions

Conditions allow you to make decisions in JavaScript code. You can use the following statements:

  • if: Executes a block of code if a specified condition is true.
  • else if: Specifies a new condition to test if the first condition is false.
  • else: Executes code if all the preceding conditions are false.
  • switch: A more efficient alternative to multiple if statements.

Examples

Below are some practical examples of JavaScript concepts:

Example: Variables

This example demonstrates the usage of var, let, and const.


let message = "Hello, welcome to learning JavaScript!";
const greeting = "Welcome!";
var info = "JavaScript makes websites interactive and dynamic.";

function displayMessage() {
    console.log(message);
    console.log(greeting);
    console.log(info);
}
Example: Functions

This example demonstrates how to declare and call functions.


function greetUser(name) {
    console.log("Hello, " + name + "!");
}

greetUser("Andri");
